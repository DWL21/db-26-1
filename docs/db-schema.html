subscribers

구독자 계정 테이블

컬럼 타입 키 Nullable 기본값 설명
id BIGINT PK NO autoincrement 구독자 고유 식별자
email VARCHAR UQ NO — 구독자 이메일 주소 (소문자 정규화)
unsub_token VARCHAR UQ YES — 구독 해지용 토큰 (secrets.token_hex(24))
created_at TIMESTAMP — NO now() 구독 등록 시각

subscriptions

구독자별 카테고리 매핑 테이블

컬럼 타입 키 Nullable 기본값 설명
id BIGINT PK NO autoincrement 레코드 고유 식별자
subscriber_id BIGINT FK NO — subscribers.id 참조 · CASCADE DELETE
category VARCHAR — NO — 구독 카테고리명 (학사 / 장학 / 채용 등)
created_at TIMESTAMP — NO now() 카테고리 구독 등록 시각
UNIQUE CONSTRAINT uq_subscriber_category (subscriber_id, category)

auth_codes

이메일 인증번호 임시 저장 테이블 · 인증 완료 시 즉시 삭제

컬럼 타입 키 Nullable 기본값 설명
email VARCHAR PK NO — 인증 요청 이메일 (이메일당 1개 유지)
code VARCHAR — NO — 6자리 인증번호
expires_at TIMESTAMPTZ — NO — 인증번호 만료 시각 (요청 시각 + 5분)

notices

발송된 공지 링크 기록 테이블 · 중복 발송 방지용

컬럼 타입 키 Nullable 기본값 설명
id BIGINT PK NO autoincrement 레코드 고유 식별자
link VARCHAR UQ NO — 공지 원문 URL · 신규 판정 기준
created_at TIMESTAMP — NO now() 링크 최초 저장(발송 성공) 시각
